# Datasets-Results
/TrainingDatasets: Excel files of the '60x60 Square Luminaires', 'Downlight Luminaires', 'Linear Luminaires', 'LinearIndirect Luminaires' prediciton models training datasets. "*.X_Full" files are physical units, "*.X_Full_Norm" files are normalized [0,1].
/LuminaireDatas: Luminaire datas used in the dataset creation and prediction model. Also have some calculation results of luminaires, calculated in simulation program DIALux.
/TrainingMetrics: Training metrics as *.csv files and training curve image files of all of the test in the paper. (4 Different Architecture)x(2 Different Activation Function)x(4 Luminaire Types)= 32 Folders 

Other details can be found at papar titled: "Application of a Machine Learning-Based Prediction Model for the Design of General Lighting Systems in Offices"
